Запуска проекта:
1) Установка зависимостей. Установите необходимые зависимости, используя pip: 
pip install fastapi uvicorn sqlalchemy pydantic

2) Настройте базы данных. Измените строку подключения к базе данных в файле database.py, если ваши настройки отличаются:
  
URL_DATABASE = 'postgresql://postgres(логин):postgres(пароль)@localhost:5422/EducationStud(схема)'

3) Запустите сервер с помощью консольной команды python main.py либо с помощью uvicorn: uvicorn main:app --reload

Сервер будет запущен на http://127.0.0.1:8000/docs
